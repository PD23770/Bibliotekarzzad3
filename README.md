Biblioteka — rozszerzenie z wykorzystaniem programowania funkcyjnego.

Program umożliwia:

= wyszukiwanie książek po frazie w tytule lub autorze,
= filtrowanie tylko dostępnych książek,
= sortowanie książek według: tytułu, autora, liczby dostępnych sztuk,
= wypożyczanie książek,
= zwracanie książek,
= rezerwację niedostępnego tytułu,
= wysyłanie próśb o przedłużenie wypożyczenia,
= sprawdzanie przez bibliotekarza, czy przy danej książce istnieje rezerwacja,
= wyświetlanie statystyk bibliotekarza.
